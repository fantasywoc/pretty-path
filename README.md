# pretty-path


directory example:

```
# 示例目录结构字典
directory_structure = {
    'documents': {
        'work': {
            'report.docx': None,
            'presentation.pptx': None
        },
        'personal': {
            'photos': {
                'summer.jpg':None,
                'winter.png':None
            },
            'diary.txt':None,
            'Path': {}
        }
    }
}
```
pretty-path example:

```
└──·documents
    └──·work
        └── report.docx
        └── presentation.pptx
    └──·personal
        └──·photos
            └── summer.jpg
            └── winter.png
        └── diary.txt
        └──·Path

```
